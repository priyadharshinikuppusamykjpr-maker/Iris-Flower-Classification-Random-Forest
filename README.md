Iris Flower Using Random Forest
Project Overview:
This project classifies Iris flowers into one of three species—Setosa, Versicolor, or Virginica—using the Random Forest Classifier. It demonstrates a complete machine learning workflow, including data preprocessing, feature engineering, model training, evaluation, and prediction.

 Features:
 Iris flower dataset created using Pandas
Feature Engineering (Petal_Area)
Train-Test Split
Random Forest Classification
Model Evaluation
Prediction for a new flower
 Technologies Used:
 Python
Pandas
Scikit-learn

Dataset Features:
Sepal Length
Sepal Width
Petal Length
Petal Width
Petal Area (Engineered Feature)

 Target Classes:
 Setosa
Versicolor
Virginica

 Machine Learning Workflow:
 Create the Iris dataset.
Perform feature engineering.
Split the dataset into training and testing sets.
Train the Random Forest Classifier.
Predict the flower species.
Evaluate the model using accuracy and classification metrics.

 Evaluation Metrics:
 Accuracy Score
Confusion Matrix
Classification Report
Precision
Recall
F1-Score

 Prediction:
The trained model predicts the species of a new Iris flower based on its sepal and petal measurements.
